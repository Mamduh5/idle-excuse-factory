# Idle Excuse Factory

Mobile-first Phaser 3 idle game prototype about running a tiny excuse factory.

## Getting Started

```bash
npm install
npm run dev
npm run check
```

- `npm install` installs project dependencies.
- `npm run dev` starts the local Vite development server.
- `npm run check` runs system tests, TypeScript checks, and the production build.

## Development Commands

- `npm run dev` starts the local Vite development server.
- `npm run check` runs the current non-browser release checks: system tests, TypeScript typecheck, and production build.

Run `npm run check` before committing changes or asking Codex for the next implementation pass.
